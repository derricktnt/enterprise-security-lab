# Linux Systems Administration Labs

This module demonstrates enterprise Linux administration across installation, authentication, firewalling, SELinux enforcement, logging, auditing, and system lifecycle management.

## Topics Covered
- RHEL installation and baseline configuration
- User authentication and privilege delegation
- SELinux and firewalld enforcement
- System auditing with auditd
- Backup and disaster recovery workflows

## Outcomes
- Secure Linux baseline suitable for production deployment
- Fully documented system hardening and validation procedures
